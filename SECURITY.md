# Security

Connector tokens are generated by the Piwigo instance and stored as SHA-256
hashes in Piwigo's `_data` directory.

The Canva app never receives Piwigo passwords or Piwigo API keys.

If a connector token is exposed, revoke it from:

```text
https://your-piwigo.example.com/plugins/canva_connector/connect.php
```

Then generate a new token.
